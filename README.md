# baskerville_plugin
Baskerville Wordpress Plugin

## Building

```commandline
cd ..
zip -r baskerville-plugin.zip baskerville_plugin/ -x "*.DS_Store" "baskerville_wp/.idea/*" "baskerville_wp/.git/*" "baskerville_wp/node_modules/*" "baskerville_wp/*.log"  
```

## Installation
Upload and activate baskerville-plugin.zip in your Wordpress plugins admin page.
